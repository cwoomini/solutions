# [(1000) A+B](https://www.acmicpc.net/problem/1000)

| Time Limit | Memory Limit |
| :--------: | :----------: |
|     2s     |    128 MB    |

## Question

두 정수 A와 B를 입력받은 다음, $A+B$를 출력하는 프로그램을 작성하시오.


## Input

첫째 줄에 A와 B가 주어진다. $(0 < A, B < 10)$


## Output

첫째 줄에 $A+B$를 출력한다.

## Examples
```
[Input]
1 2

[Ouput]
3
```
